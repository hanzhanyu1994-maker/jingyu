# 🐋 鲸语

AI 对话与用量，一触即达。

鲸语是一款轻量的鸿蒙快捷工具，帮你快速访问 AI 平台的用量后台和对话页面，无需每次打开浏览器。

## ✨ 功能

- **用量看板** — 一键查看余额、消费和用量统计
- **AI 对话** — 直达网页版对话，享受完整功能
- **沉浸设计** — 原生首页，毛玻璃顶栏，流畅渐入动画
- **隐私优先** — 不收集任何个人信息，无需注册账号

## 📱 系统要求

- HarmonyOS 6.1.0 及以上
- 手机 / 平板
